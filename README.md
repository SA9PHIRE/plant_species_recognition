# 🌿 Plant Species Recognition

Учебный проект по распознаванию видов растений на фотографиях с помощью нейросети.

## 🎯 Что делает проект

Загружаете фото цветка → нейросеть определяет, что это за растение.

## 📁 Структура проекта

| Файл/папка | Что делает |
|------------|-------------|
| `scripts/download_dataset.py` | Скачивает датасет с Kaggle |
| `src/train.py` | Обучает нейросеть |
| `src/predict.py` | Распознаёт растение на новом фото |
| `src/model.py` | Архитектура нейросети (ResNet) |
| `src/dataset.py` | Загружает и обрабатывает изображения |
| `src/utils.py` | Вспомогательные функции |
| `requirements.txt` | Список нужных библиотек |
| `LICENCE` | - |
| `.gitignore` | что НЕ загружать на GitHub |


## 🚀 Запуск проекта

### 1. Установить зависимости

pip install -r requirements.txt

### 2. Скачать датасет с цветами
python scripts/download_dataset.py

### 3. Обучить нейросеть
python src/train.py

### 4. Проверить на своем фото
python src/predict.py --image моё_фото.jpg
